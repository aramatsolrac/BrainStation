## Overview

### The Exercise
- Build a multi-page website to a new local coffee shop which is set to launch soon.

### Features
- Multi-page website
- Responsive design
- Users can access basic information on landing page
- Users can access further details on menu page

### Technologies

- HTML5
- CSS
- BEM
- FlexBox
